# 🌱 Sistema de Cadastro de Espécies em Risco (SCER)

Este projeto foi desenvolvido para cadastrar, consultar e gerenciar espécies da fauna e flora ameaçadas de extinção. Inspirado na missão de instituições de conservação (como o ICMBio), o projeto evoluiu de um script simples para uma **Aplicação Web Completa (Single Page Application)**, focada em performance, design moderno e usabilidade.

## 🚀 Tecnologias utilizadas
- **Backend:** Python 3, Flask (API RESTful)
- **Banco de Dados:** SQLite (com Prepared Statements anti-SQL Injection)
- **Frontend:** HTML5, Vanilla JavaScript, TailwindCSS (Design Responsivo e Glassmorphism)
- **Ferramentas:** Git, GitHub, VS Code

## ✨ Funcionalidades
- **Dashboard Dinâmico:** Interface "Single Page Application" com transições suaves.
- **Gestão Completa (CRUD):** Cadastrar, Listar, Editar e Remover espécies.
- **Upload de Imagens:** Suporte a salvamento de fotos reais para cada espécie diretamente no servidor local.
- **Filtro Analítico:** Buscas precisas por Grau de Ameaça (IUCN) com geração de estatísticas em tempo real.
- **Busca Rápida:** Escaneamento direto pelo ID da espécie.
- **Segurança:** Proteção contra XSS e SQL Injection ativa.

## 🎯 Objetivo
Aplicar conhecimentos técnicos de forma prática (Full-Stack), desde a estruturação do banco de dados relacional até a criação de uma API REST e consumo por uma interface gráfica moderna e responsiva, desenvolvendo um portfólio sólido na área de T.I.

## ⚙️ Como rodar o projeto localmente

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/Dimmy-dev/-SCER-.git
   cd -SCER-/src
   ```

2. **Instale as dependências necessárias:**
   ```bash
   pip install -r requerimentos.txt
   ```

3. **Inicie o servidor Web:**
   ```bash
   python main.py
   ```

4. **Acesse no navegador:**
   Abra `http://127.0.0.1:5000` para ver o sistema rodando!
